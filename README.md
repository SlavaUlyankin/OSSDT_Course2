# LSB Steganography Application

## Описание
Приложение для встраивания и извлечения текстовой информации в цифровые изображения методом LSB (Least Significant Bit).

## Требования
- Python 3.10+
- Все зависимости указаны в requirements.txt

## Установка

1. Создать виртуальное окружение:
```bash
python -m venv venv
```

2. Активировать виртуальное окружение:
- Windows: `venv\Scripts\activate`
- Linux/Mac: `source venv/bin/activate`

3. Установить зависимости:
```bash
pip install -r requirements.txt
```

## Запуск приложения
```bash
python gui_main.py
```

## Использование

### Встраивание информации
1. Откройте вкладку "Встраивание"
2. Выберите изображение-контейнер (PNG или BMP)
3. Введите текст для встраивания
4. Нажмите "Встроить сообщение"
5. Сохраните результат

### Извлечение информации
1. Откройте вкладку "Извлечение"
2. Выберите стего-изображение
3. Нажмите "Извлечь сообщение"
4. Просмотрите результат

## Структура проекта
- lsb_encoder.py - встраивание информации
- lsb_decoder.py - извлечение информации
- metrics.py - расчет метрик качества
- gui_main.py - графический интерфейс
- requirements.txt - зависимости
- README.md - документация
